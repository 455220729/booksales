# 基于java的线上书刊销售管理系统

## 一、系统介绍

后台管理员登录包含以下功能：

- 权限管理，店铺管理，商品管理，订单管理，用户管理

前台功能：

- 注册，登录，
- 浏览商品、下单

## 二、环境介绍

系统环境：windows10

基础环境 ：eclipse、javaSE-1.7(jre)、MySQLl5.7、Maven3.9.5、Navicat for MySQL、Tomcat7.0
